:::anchor 快捷键

单元格选择支持以下快捷键（参考 excel 快捷键）：

| 功能               | 快捷键            |
| :----------------- | :---------------- |
| 向上移动活动单元格 | `↑`               |
| 向右移动活动单元格 | `→`               |
| 向下移动活动单元格 | `↓`               |
| 向左移动活动单元格 | `←`               |
| 向下移动活动单元格 | `Enter`           |
| 向上移动活动单元格 | `Shift` + `Enter` |
| 向右移动活动单元格 | `Tab`             |
| 向左移动活动单元格 | `Shift` + `Tab`   |
