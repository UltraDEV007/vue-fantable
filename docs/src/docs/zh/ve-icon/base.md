:::anchor 使用

:::demo 1、`name`属性设置图标的名称<br> 2、`size`属性设置图标的大小<br> 3、`color`属性设置图标的颜色

```html
<template>
    <div>
        <ve-icon name="double-right-arrow" />
        <ve-icon name="double-right-arrow" color="red" />
        <ve-icon name="double-right-arrow" :size="40" />
    </div>
</template>
```
